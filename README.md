# مصحف ورش الرقمي (Warsh Digital Mushaf)

تطبيق ويب تفاعلي عالي الأداء ومُتجاوب لقراءة والاستماع إلى القرآن الكريم برواية ورش عن نافع.

## المميزات (Features)

- **صفحات مصحف تفاعلية (Vector-based)**: الاستفادة من مشروع `quran-svg` لتقديم صفحات مصحف عالية الدقة وقابلة للتفاعل.
- **التلاوة الصوتية المتزامنة**: تشغيل تلاوات صوتية مع حلول مبتكرة للتعامل مع اختلافات ترقيم الآيات بين الروايات (Hafs vs Warsh) باستخدام أدوات مثل `qiraat-ayah-map`.
- **تصميم متجاوب وحديث**: واجهة مستخدم توفر تجربة ممتازة على أجهزة الجوال وأجهزة الكمبيوتر مع تصميم عصري وسريع.
- **مشاركة الآيات كصورة**: إمكانية مشاركة آية محددة كصورة عالية الجودة بضغطة زر.

## التقنيات المستخدمة (Tech Stack)

- HTML, CSS, JavaScript (Vanilla JS)
- Tailwind CSS (عبر CDN)

## طريقة التشغيل (Setup & Installation)

هذا المشروع تطبيق ويب ثابت (Static Web App)، ويعمل مباشرة بدون الحاجة لأي بيئة عمل أو تثبيت حزم.

1. قم بتحميل المشروع:
   ```bash
   git clone https://github.com/alinice1998/warsh-digital-mushaf.git
   ```

2. بكل بساطة، قم بفتح ملف `index.html` في متصفحك للبدء، أو قم برفعه على أي خادم استضافة ثابت (مثل GitHub Pages، Vercel، أو XAMPP).

## ملاحظات
يعتمد هذا المشروع على مزامنة دقيقة للتلاوات وضبط الفروقات بين القراءات المختلفة لضمان تجربة استماع دقيقة ومتوافقة تماماً مع المصحف المكتوب برواية ورش.
